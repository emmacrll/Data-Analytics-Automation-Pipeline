# 💡 Insights Business - Superstore Analysis

## Objectif

Cette section présente les principaux enseignements obtenus grâce à l’analyse SQL et au dashboard Power BI.

L’objectif est d’identifier les facteurs influençant la performance commerciale : clients, produits, segments et régions.

---

# 📊 Performance globale

L’analyse du dashboard permet d’obtenir une vision générale de la performance commerciale.

Les principaux indicateurs suivis sont :

* Chiffre d’affaires total (Total Sales)
* Profit total (Total Profit)
* Nombre de commandes (Total Orders)
* Nombre de clients (Total Customers)
* Marge bénéficiaire (Profit Margin)

Ces indicateurs permettent d’évaluer la santé globale de l’activité et de suivre son évolution.

---

# 👥 Analyse des segments clients

L’analyse des segments clients permet de comparer la contribution des différents groupes :

* Consumer
* Corporate
* Home Office

Les objectifs de cette analyse sont :

* Identifier le segment générant le plus de chiffre d’affaires
* Comparer la rentabilité entre les segments
* Comprendre quels clients apportent le plus de valeur à l’entreprise

### Insight principal

Le segment générant le plus de ventes n’est pas forcément celui générant le plus de profit.

L’analyse ventes vs profit permet donc d’identifier les segments les plus intéressants d’un point de vue business.

---

# 📦 Analyse des produits

L’analyse produit permet d’identifier les catégories et produits ayant le plus d’impact sur la performance.

Les analyses réalisées :

* Produits avec le plus grand chiffre d’affaires
* Produits générant le plus de profit
* Produits avec une rentabilité faible ou négative

### Insight principal

Certains produits peuvent générer un volume important de ventes mais une faible rentabilité.

Il est donc important d’analyser le profit et non uniquement le chiffre d’affaires afin d’optimiser les décisions commerciales.

---

# 🌎 Analyse géographique

L’analyse par région permet d’identifier les zones commerciales les plus performantes.

Les objectifs :

* Comparer les ventes par région
* Identifier les régions contribuant le plus au profit
* Détecter les zones moins performantes

### Insight principal

La performance commerciale varie selon les régions. Une analyse géographique permet d’identifier les opportunités de croissance et les zones nécessitant une amélioration.

---

# 👤 Analyse des clients

L’analyse des clients permet d’identifier :

* Les clients générant le plus de chiffre d’affaires
* Les clients les plus profitables
* Les différences entre volume d’achat et rentabilité

### Insight principal

Les clients ayant les plus grandes dépenses ne sont pas toujours les plus rentables. La valeur client doit être évaluée selon le profit généré et non uniquement selon les ventes.

---

# 🎯 Recommandations business

À partir des analyses réalisées, plusieurs actions peuvent être envisagées :

* Favoriser les produits avec une forte rentabilité
* Réévaluer les produits générant des pertes
* Développer les segments clients les plus profitables
* Analyser les raisons des faibles performances régionales
* Suivre régulièrement les KPI grâce au reporting automatisé

---

# Conclusion

L’analyse Superstore montre l’importance de combiner plusieurs dimensions :

* Performance financière
* Analyse produits
* Analyse clients
* Analyse géographique

L’utilisation combinée de SQL, Power BI et Power Automate permet de transformer les données brutes en informations utiles à la prise de décision.
